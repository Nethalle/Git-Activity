# Git Activity Repository

## Description
This repository was created for the Git and GitHub activity in CS223 - Application Development and Emerging Technologies.

The purpose of this activity is to practice creating repositories, using basic Git commands, managing branches, and synchronizing local and remote repositories.

---

## Objectives
- Create a GitHub repository
- Synchronize local and remote repositories
- Practice basic Git commands
- Create and manage branches
- Push and pull updates using Git

---

## Tools Used
- Git
- GitHub
- Git Bash / Terminal
- Visual Studio Code

---

## Activity Steps Performed
1. Created a GitHub repository
2. Initialized local repository
3. Linked local repository to GitHub
4. Added and committed files
5. Pushed files to GitHub
6. Created and switched branches
7. Merged branches into main
8. Pulled updates from remote repository
